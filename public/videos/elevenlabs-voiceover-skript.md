# ElevenLabs – Voiceover-Skript & Aussprache (KI-Tools 2026)

Zwei Wege zur korrekten Aussprache englischer Begriffe:

- **Weg A (empfohlen, kugelsicher):** Das **Lautschrift-Skript** unten direkt in ElevenLabs einfügen. Funktioniert mit jedem mehrsprachigen Modell, ohne Dictionary.
- **Weg B (für die Zukunft):** Die Datei `elevenlabs-aussprache.pls` als **Pronunciation Dictionary** hochladen und das **Originalskript** verwenden. Dann musst du nie wieder umschreiben.

**Empfohlene Settings:** Stimme = deine geklonte Stimme · Modell = **Eleven Multilingual v2** (oder v3) · Stability ~50 % · Similarity ~75 %. Erst eine Zeile testen, Aussprache prüfen, dann alles generieren.

## Dictionary in ElevenLabs Studio hinterlegen (Kurzfassung)

1. ElevenLabs → **Studio** → neues Projekt anlegen, Originalskript einfügen.
2. Stimme = deine geklonte Stimme, Modell = **Eleven Multilingual v2**.
3. Button **„Open pronunciations editor"** öffnen.
4. **„Open all pronunciation dictionaries"** → **Upload** → `elevenlabs-aussprache.pls` wählen.
5. Dictionary wird automatisch mit dem Projekt verbunden. Mit **Play** einzelne Wörter testen.
6. Generieren, Audio exportieren – fertig.
